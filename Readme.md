# Convoy Game

Небольшая игра военного конвоя, доставляющего ценный груз под охраной военных кораблей.


![main_view](https://github.com/xioazhweng/Convoy-game/blob/main/preview/1.png?raw=true)

![game_view](https://github.com/xioazhweng/Convoy-game/blob/main/preview/2.png?raw=true)

## Возможности

- Покупка кораблей и вооружения в рамках бюджета
- Перевозка груза со снижением скорости в зависимости от загрузки
- Боевая система с учётом боезапаса оружия
- Атаки пиратов с баз вдоль маршрута

## Сборка и запуск

```bash
mkdir build
cd build
cmake ..
make
./game
```

## Конфигурация

Загрузка параметров миссии, типов вооружения, кораблей и моделей пиратов из конфигурационных файлов.
